# Changelog

## [0.2.0] - 2026-05-13

### Added

- `mdpdf import-theme <css>` for importing Typora CSS themes as Typst themes.
- `mdpdf themes` for listing built-in and imported themes.
- Imported Typora-inspired theme files, including `purple-typora`.
- MIT license file and public repository metadata.

### Improved

- Markdown tables generated by Pandoc are unwrapped from table figures so Typst
  can paginate them across pages naturally.
- Imported H1 styles now use a full-width, subtle rule instead of an awkward
  short underline.
- `--paper` now propagates into Typst templates.
- `--paper letter` is normalized to Typst's `us-letter` paper name.
- README expanded with installation, theme import, configuration, and
  development instructions.

## [0.1.0] - 2026-05-12

### Added

- `mdpdf` CLI command.
- `minimal` theme: clean, print-friendly, gray code blocks.
- `purple` theme: purple accent color, dark code blocks, inspired by Typora's
  purple theme.
- Pandoc-based Markdown to Typst conversion.
- Automatic table of contents.
- Page headers and footers with page numbers.
- `<!-- pagebreak -->` manual page break support.
- Title, author, and date display.
- Dependency checks with clear error messages.
- `--keep-typ` flag for debugging the intermediate Typst file.
- `mdpdf.config.json` config file support.
- Example documents: `basic.md` and `technical-report.md`.
