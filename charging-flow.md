# 54.6V / 2A 充电逻辑流程

本工程使用 WY8S9003 平台的 ADC、PWM、Timer、GPIO、LED 与调度框架。当前硬件最大输出电流为 2A，软件充电目标为 `54.6V / 2A`。

## 核心状态


| 状态                | 行为                                                                                        |
| ----------------- | ----------------------------------------------------------------------------------------- |
| `STATE_INIT`      | 关闭继电器，Vset 进入 0% 占空比输出 54.6V，Iset 限到 300mA，清计数后进入等待电池                                            |
| `STATE_WAIT_BAT`  | 绿灯先常亮 3s，再慢闪等待电池；继电器断开时 Vset 常态为 0% 占空比；带接入确认和 AC 掉电斜坡锁定               |
| `STATE_PRECONNECT` | 红灯长亮 2s，继电器断开；先将 `Vset` 渐变到 `Vbat` 附近，末段回到 0% 占空比探测电池/负载是否仍接入，断开则立即回空载输出并绿灯慢闪 |
| `STATE_CHARGING`  | 继电器闭合，Vset 立即回到 0% 占空比输出 54.6V，Iset 斜坡升到 2A；用真实电流历史识别拔出、BMS 关 MOS 与满充 |
| `STATE_FULL`      | 满充，断开继电器，Vset 回到 0% 占空比输出 54.6V，Iset 限到 300mA，绿灯常亮；带 AC 掉电斜坡锁定，电池电压重新低于满充阈值并确认后再充           |
| `STATE_OVP`       | 过压保护，断继电器，红灯快闪；电压恢复 3s 后回初始化                                                              |
| `STATE_OCP`       | 过流保护，断继电器，红灯快闪；电流恢复 3s 后重新进入预连接确认                                                          |
| `STATE_SCP`       | 短路保护，断继电器，红灯快闪；电压恢复 3s 后重新进入预连接确认                                                          |
| `STATE_ADC_FAULT` | ADC 连续异常保护，断继电器，红灯快闪；ADC 正常 3s 后回初始化                                                      |
| `STATE_OTP`       | 过温保护，断继电器，红灯快闪；温度恢复并确认后重新进入预连接确认                                                           |
| `STATE_TIMEOUT`   | 充电超时锁定，断继电器，红灯快闪                                                                          |


## 关键阈值


| 项目           | 当前值                                                   |
| ------------ | ----------------------------------------------------- |
| 允许启动电压       | `25V <= Vbat < 53V`                                   |
| 满充电压窗口       | `53V <= Vbat < 56V`                                   |
| 输出目标         | `54.6V / 2A`                                          |
| 预连接限流        | `300mA`                                               |
| 满充电流         | `< 300mA` 持续 `1s`                                     |
| 大电流拔出阈值      | 历史电流峰值 `> 600mA` 且快速跌到 `< 300mA`                        |
| BMS 关 MOS 判满 | 历史电流峰值 `<= 600mA` 且当前 `< 100mA`                       |
| 短路           | 充电中 `Vbat < 10V` 连续 `300ms`                               |
| 过压           | `Vbat >= 56V` 连续 `300ms`                              |
| 过流           | `Iout > 2.4A` 持续 `1s`                                 |
| 过温           | NTC ADC 原始值 `<= 405` 连续 `300ms`，恢复到 `>= 2861` 持续 `3s` |
| 充电超时         | `720000` 个 100ms tick                                 |


## Vset 与采样说明

- 上电和继电器断开时，Vset 默认保持 0% 占空比（低电平），使充电器输出 54.6V。
- 检测到电池接入后，预连接阶段才将 Vset 从 0% 渐变到接近 Vbat，末段回到 0% 占空比做开路复核；只有电池/负载仍把瞬时 Vbat 与滤波 Vbat 都拉在充电窗口内才吸合继电器。
- 预连接取消时会立即把 Vset 拉回 0% 占空比，并短暂屏蔽重新接入判定，避免 PWM 斜坡和 Vbat 滤波尾巴造成红绿灯反复跳变。
- 继电器吸合后，Vset 立即回到 0% 占空比，使电压环回到 54.6V。
- Vset 空载实测端点：0% 占空比输出 54.6V，100% 占空比输出 42.7V。线性公式为 `Vout = 54.6V - 11.9V * duty_ratio`，反算占空比为 `duty_ratio = (54.6V - Vtarget) / 11.9V`。
- 固件实现中不使用浮点数：ADC 缓存统一为 `mV / mA` 整数，Vset/Iset 换算使用定点常量和移位近似。
- ADC 单通道每轮采样 10 次，去掉最大值和最小值后，对剩余 8 次右移平均；任意一次采样超时则丢弃本轮数据并保持上一轮缓存。
- Vout 保持采样和调试输出，但当前不参与状态判断或保护。
- AC 掉电斜坡锁定使用最新一次未滤波 Vbat 判断，并覆盖 `STATE_WAIT_BAT` 与 `STATE_FULL`；正常接入电池会瞬间拉低 Vbat，不会被 EMA 滤波误判为缓慢掉电。
- 关键状态切换会立即同步 LED 模式，避免状态已变更但灯态仍停留在上一状态。
- `MS_TO_TICKS(...)` 返回 `uint8_t`，单个确认时间不要配置超过 `25500ms`。
